The Buying Zone is a Desktop based Store Management system. 
It's a DBMS desktProject using C# and ADO.NET (SQL Server as backend Database)
Functionalities include:
  1. Managing inventory
  2. Role-based authentication and authorization
  3. Dynamic filtering
  4. Billing module
  5. Reporting and Analytics including charts and graph using SQL server Views
  6. Implemented Stored Procedures and Triggers
